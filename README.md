# Titanic Survival Analysis

## Overview
An exploratory data analysis of 891 Titanic passengers to identify 
the demographic factors that most significantly influenced survival probability.

## Key Findings
- Only 38.38% of passengers survived the disaster
- Gender was the dominant survival factor — 74.2% of women survived vs 18.89% of men
- 1st class passengers had a 62.96% survival rate compared to 24.24% in 3rd class
- Children had the highest survival rate at 57.97%
- Cross-tabulation revealed 3rd class females (50%) outlasted 1st class males (36.89%)

## Tools Used
- Python
- Pandas — data cleaning and analysis
- Matplotlib & Seaborn — visualisations
- SciPy — statistical analysis

## Files
- `titanic_analysis.py` — full analysis code
- `train.csv` — dataset from Kaggle Titanic competition
- `Titanic_Analysis_Report.pdf` — written report with findings

## How to Run
1. Install dependencies: `pip install pandas matplotlib seaborn scipy`
2. Download `train.csv` from Kaggle Titanic competition
3. Run `titanic_analysis.py`
